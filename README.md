# Ćwiczenia z obróbki i analizy danych

### 1. Środowisko wirtualne języka Python
```shell
# dla systemu Windows
# tworzymy w folderze roboczym środowisko wirtualne o nazwie my_env:
python -m venv my_env

# aktywujemy je
cd my_env
cd Scripts
activate

# przechodzimy z powrotem do folderu roboczego
cd .. 
cd .. 
```
